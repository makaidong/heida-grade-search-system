黑大绩点查询系统
======

功能
------
实现黑大所有人的成绩和绩点查询，解决传统的黑大信息门户的经常卡死而且界面有缺陷和没有计算绩点的功能。
账号为黑大人的学号，密码和信息门户中的密码一样

使用说明
------
进入到登录界面，输入账号密码即可。
